🌐 Currículo Digital Interativo — Jhonatan Alcântara de Oliveira
📖 Contexto

Este projeto apresenta um currículo digital moderno e interativo, desenvolvido em HTML, CSS e JavaScript puro.
Foi criado para demonstrar tanto o perfil profissional quanto as habilidades técnicas de Jhonatan Alcântara de Oliveira, unindo design limpo, efeitos visuais dinâmicos e responsividade para uso em qualquer dispositivo.

🚀 Funcionalidades

🎨 Design Responsivo e Minimalista — Adaptado para desktop e mobile.

🧭 Navegação suave (scroll animado) — Experiência fluida entre seções.

💫 Animação com partículas dinâmicas — Canvas interativo em segundo plano (particles-canvas).

🖱️ Cursor personalizado duplo — Interação moderna e visualmente atraente.

📑 Download direto do currículo (DOCX) — Link rápido e funcional.

👁️ Efeito de revelação no scroll — Elementos animam suavemente ao entrar na tela.

📱 Menu mobile retrátil — Interface intuitiva e compacta em telas menores.

🧩 Estrutura do Projeto
📁 CV-Interativo/
│
├── index.html        # Estrutura principal e conteúdo do currículo
├── style.css         # Estilos e layout (com design responsivo)
├── script.js         # Efeitos interativos e animações (partículas, cursor, scroll)
└── /img              # Pasta de imagens (foto de perfil e ícones, se houver)

💻 Tecnologias Utilizadas
Tecnologia	Função Principal
HTML5	Estrutura semântica do currículo
CSS3	Estilização, animações e responsividade
JavaScript (ES6)	Efeitos dinâmicos, partículas e interações do usuário
Font Awesome 6.4	Ícones modernos e leves
Google Fonts (Poppins & Oswald)	Tipografia profissional
Particles.js (CDN)	Efeito de partículas animadas no background
⚙️ Como Executar o Projeto Localmente

Clone ou baixe o repositório:

git clone https://github.com/SEU_USUARIO/CV-Interativo.git


Abra a pasta do projeto:

cd CV-Interativo


Execute localmente:

Clique duas vezes em index.html
ou

Rode um servidor local:

npx live-server


O currículo estará disponível em:

http://localhost:8080

✨ Personalização

Substitua a imagem em /img/Jhon.jpg pela sua própria foto.

Atualize o link do botão “Baixar CV” em:

<a href="doc/Curriculum  Jhonatan[1]-2.docx">


Edite as informações pessoais diretamente na seção <header> e os dados profissionais nas seções <section>.

🧠 Boas Práticas Implementadas

Código seguindo padrões semânticos HTML5.

Estilos com variáveis CSS (:root) para facilitar manutenção.

JavaScript modularizado com classes e funções reutilizáveis.

Responsividade e acessibilidade otimizadas para mobile.

Animações suaves baseadas em Intersection Observer API (melhor performance).
